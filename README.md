1 BACKEND

- Requisitos previos

Antes de iniciar, asegúrate de tener instalado:

* Node.js (v18 o superior)
* Docker y Docker Compose
* Git

- Levantar PostgreSQL con docker compose

''bash
cd backend
docker-compose up

- Ejecutar backend(node+express)

''bash
cd backend
npm install
node app.js

2 FRONTEND

''bash
cd frontend
npm install
npm run dev
